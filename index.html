Background

I’m Murilo, a Software Engineer based in Curitiba, Brazil. I hold a bachelor's degree in electrical engineering and have extensive experience in embedded software and electronics. My background in embedded systems has sharpened my skills in memory optimization, hardware interface design and low-level programming.

From a young age, I was fascinated by games like Megaman X, Super Mario Bros., and Sonic the Hedgehog, spending hours playing retro classics. My passion for gaming brings back fond memories of playing PlayStation, Xbox, and emulators with my brother and cousins. I’ve always appreciated how video games connect people, creating a shared experience of fun and engagement.

As a Game Developer, I’ve contributed to the development of the Evoverse game, using Godot to build systems such as battle mechanics, card interactions, sound, and various user interfaces like loading screens, map views, battle arenas, and pop-ups. Additionally, I’ve mentored junior developers through code reviews. Outside of my professional work, I’ve participated in two Global Game Jams, creating Crops’n Cats (runner-up) and Veggie Vendetta (4th place).

Game Jams

Veggie Vendetta was made in 48 hours in Global Game Jam 2024. The game is a multiplayer battle arena which consists of getting some weapons and pushing your opponent to barbecue fire.

I developed some features like battle system, camera scripts, dubbing some noises. I also led the team and helped with game design concepts. The art was inspired in Cult of the Lamb.

Stacks: Unity 2D, C#, Git

Refs: Art, YouTube

Crops’n Cats was a game made in 48 hours in Global Game Jam 2023, the runner up of 70 games. The game is a multiplayer farming game in which players have to clean, crop, water and seeding the terrain.

I worked in the score and highscore system, implemented sounds, created animations and helped game concepts. The cat and fluffy music theme was proposite to catch the audience's attention.

Stacks: Unity 2D, C#, Git

Refs: Art, YouTube

Worked Projects

EVOVERSE is a turn-based Blockchain strategy game. It takes place in a futuristic universe where robots have become conscious and have begun a migratory process across the vastness of the universe.

I worked developing card system, loading screens, map view, battle arena, pop ups, team builder, GPU and CPU optimization using Game Profiler.

Stacks: Godot, GDScript, SVN

Refs: Website, YouTube
